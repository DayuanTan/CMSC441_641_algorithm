### 2.5.5 Fibonacci Number

F(N) = F(N-1) + F(N-2)

F(0)= F(1) = 0

![](./fibonacci.jpg)

Pseudocode:
```python
F(N) # store all results of sub-questions
for i = 1 ... N
    if i == 0 or 1
        return 0
    else 
        return F(i-1) + F(i-2)
```
